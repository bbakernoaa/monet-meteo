## [Unreleased]

### Added
- Initial release of monet-meteo package
- Thermodynamic calculations module with functions for potential temperature, equivalent potential temperature, virtual temperature, saturation vapor pressure, mixing ratio, and lapse rates
- Derived parameters module with functions for heat index, wind chill, lifting condensation level, and wet bulb temperature
- Dynamic calculations module with functions for vorticity, divergence, geostrophic wind, gradient wind, absolute vorticity, and potential vorticity
- Statistical operations module for micrometeorology calculations
- Data models for atmospheric profiles and related meteorological data
- Type hints for better code quality and IDE support
- xarray integration for multi-dimensional meteorological data
- Unit conversion utilities with Pint integration
- Coordinate transformation utilities
- Interpolation methods for meteorological data
